# Hacker news app
#### BR Group test task by Arsenii Ermolaev

## What is used

- Typescript
- React
- React Router
- MUI
- Hacker News API

## How to run

### Clone the repo

    git clone https://github.com/tigorthemusichead/react-hackernews-app.git

### Install dependencies

    cd react-hackernews-app
    npm install

### Run the app on port 3000

    npm start
